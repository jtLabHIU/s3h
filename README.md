# jtS3Helper
jtLab Scratch 3 Helper app
- work on Electron
- run in task tray
- communicate with Scratch 3 through WebSocket

## How to run
- `git clone https://github.com/jtLabHIU/s3h.git`
- `cd s3h`
- `npm install`
- `electron .`
